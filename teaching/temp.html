Utilisation du terminal

\\u003e Aide pour les commandes Unix : man (MANual).

Ouvrez un terminal. Par défaut, vous êtes dans votre espace de travail (home).

  1. Affichez les éléments contenus dans le répertoire principal (racine ~ \u003d ou "répertoire home") en utilisant la commande ls.
  2. Si le répertoire principal ne contient pas un sous-répertoire nommé \u0027Documents\u0027, créez-le en utilisant la commande mkdir.
  3. Depuis \u0027Documents\u0027, créez un sous-répertoire nommé \u0027Septembre_2018\u0027. Pour se déplacer d\u0027un répertoire à un autre, utilisez la commande cd.
  4. Créez deux fichiers \u0027tp1.py\u0027 et \u0027tp2.py\u0027 dans le dossier \u0027Septembre_2018\u0027. Pour cela, utilisez la commande touch.
  5. Quelle est la commande qui affiche à l\u0027écran le nom de votre répertoire courant ?
  6. Écrivez les commandes qui vous permettent d\u0027aller dans le répertoire principal et d\u0027afficher récursivement son contenu (sous-dossiers et fichiers).
  7. Dessinez l’arborescence de tous les répertoires et fichiers depuis le répertoire principal.
  8. Écrivez une commande qui déplace dans \u0027Documents\u0027 tous les fichiers .py du répertoire \u0027Septembre_2018\u0027. Pour cela, utilisez la commande mv.
  9. Écrivez une commande qui supprime le répertoire \u0027Septembre_2018\u0027. Pour cela, utilisez la commande rmdir.
